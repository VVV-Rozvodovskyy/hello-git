# 🎓 Основи Git та GitHub: Практична Робота

Вітаємо у базовому шаблоні для практичної роботи! Цей репозиторій створений для того, щоб ви могли безпечно потренуватися працювати з інструментами контролю версій.

## 🎯 Ваше завдання:
1. Зробити **Fork** цього репозиторію.
2. Склонувати його на свій комп'ютер через PyCharm.
3. Створити гілку `feature/my-greeting` та додати туди нові файли (`greeting.py` та `math_magic.py`).
4. Навчитися робити **Commit** та **Push**.
5. Відчути на собі, що таке **Merge Conflict** та як його вирішувати (саботаж від викладача!).
6. Створити **Pull Request**.

Удачі та хай прибуде з вами сила Git! 🚀
